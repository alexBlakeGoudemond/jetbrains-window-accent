# About Window Accent

Window Accent was created to solve a simple but common problem:

When working with multiple JetBrains IDE windows, it is easy to lose track of which instance holds which repository

Window Accent adds subtle, customizable color accents that help distinguish projects while preserving the native
JetBrains experience. In situations where 3+ instances of the IDE are up – it is much easier to identify quickly which
instance is for which repo

## Design Philosophy

- Lightweight
- Native appearance
- Zero telemetry
- Open source
- Respect user privacy

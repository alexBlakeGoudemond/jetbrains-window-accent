# Privacy Policy

Last updated: 2026-07-12

## Overview

Window Accent respects your privacy.

The plugin is designed to run entirely within your JetBrains IDE and does not collect, transmit, or store any personal
information.

## Data Collection

Window Accent does not collect:

- Personal information
- Usage analytics
- Telemetry data
- Project information
- Source code
- File contents
- IDE activity

## Network Access

Window Accent does not make network requests or communicate with external services.

All plugin functionality is performed locally within your IDE.

## Configuration Data

Plugin settings, including accent color preferences, are stored using the standard JetBrains IDE settings storage
mechanism.

These settings remain on your machine and are managed by your IDE.

## Third-Party Services

Window Accent does not integrate with third-party analytics, advertising, or tracking services.

## Open Source

Window Accent is open source and available under the MIT License.

The source code can be reviewed on GitHub.

# Window Accent

> Distinguish between multiple JetBrains IDE windows with visual cues

---

## Welcome

Window Accent helps you add a color panel, custom titles to your IntelliJ IDE windows. This is particularly useful when
there are multiple IDE Instances open at the same time.

Whether you're juggling multiple projects, environments or branches, Window Accent makes it easier to identify where
you're working at a glance.

---

## Features

- Lightweight
- Native JetBrains integration
- Custom accent colors
- Custom window titles
- Open Source (MIT)
- Available on the JetBrains Marketplace
- Available on the Github

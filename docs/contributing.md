## Contributing

If you would like to contribute - feel free to model an issue (Bugs, Feature Requests, or Questions)
or submit a pull request.

We welcome contributions of all kinds, including code, documentation, and design.

### License

By contributing, you agree that your contributions will be licensed under the
[MIT License](https://github.com/alexBlakeGoudemond/jetbrains-window-accent/blob/develop/LICENSE).

## Reporting Bugs

Before opening an issue:

- Check the existing issues
- Confirm you are using the latest version
- Include:
    - IDE version
    - Window Accent version
    - Steps to reproduce
    - Screenshots if relevant

## Feature Requests

Feature ideas are welcome!

Please describe:

- What problem does this solve?
- How would you expect it to work?
- Are there alternatives you considered?
- Visual Aids if relevant (MS Paint screenshot etc)

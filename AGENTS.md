# Agents README

This repository may make use of an ai-playbook, which is a collection of preferences and requests for the AI model to
follow. The ai-playbook can be used to guide the AI model's behavior and ensure consistency across different tasks and
use cases. This ai-playbook is defined by the user – and as such it is not tracked by Git.

For all requests from the user – please refer to [AGENTS.playbook.md](AGENTS.playbook.md), if it is present. If missing,
the ai-playbook may be missing and so you can continue as per normal. If missing, do not look for these resources -
assume they are absent.

DO NOT EDIT THE FILES IN THE AI-PLAYBOOK, DO NOT EDIT `AGENTS.playbook.md`.
